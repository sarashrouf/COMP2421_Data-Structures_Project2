# Overview
This project is a C-based text editor that allows users to load, edit, and manage text using **Undo and Redo operations**.

The application uses **stacks (Undo & Redo)** and a **queue** to track changes such as inserting and removing strings. Users can modify the text dynamically, revert changes, and save the final result to an output file.

The system demonstrates the use of fundamental data structures like **Linked Lists, Stacks, and Queues** in building a simple interactive text editor.
